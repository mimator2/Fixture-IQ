================================================================================
DATA FIELDS IN EACH CSV
================================================================================

season:       Season name (e.g., "2022-2023")
player_id:    Unique player identifier
player_name:  Player name
team_name:    Team name
injury_type:  "Missing Fixture" (confirmed out) OR "Questionable" (doubtful)
reason:       Injury type (e.g., "Thigh Injury", "Knock", "Suspension")
fixture_date: Date injury was reported (YYYY-MM-DD)
days_out:     Number of days player was injured (0 if "Questionable" only)
